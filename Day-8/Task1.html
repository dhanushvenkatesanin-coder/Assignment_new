About inline,internal and external css

Inline css

We can use style attribute inside a tag to style element.It for rare and single use but inline css have more priority than internal and external css

Internal css

We can use style tag to style a single html file. Style tag is nested in head section of the html

External css

We can use seperate css file to style a single or more html file.We can link seperate css file to html file by link tag.this method of styling html file is said as external css
